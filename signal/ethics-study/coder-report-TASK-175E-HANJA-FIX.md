### 판별 방법
파이썬 스크립트로 각 `[一-龥]+` 토큰에 대해 아래 상태를 병렬 판정:
1. 라인 내 `"` 카운트가 홀수면 → `"..."` 안 (예외)
2. 가장 최근 열린 `『` 가 `』` 보다 뒤면 → `『...』` 안 (예외)
3. 가장 최근 열린 `<!--` 가 `-->` 보다 뒤면 → HTML 주석 안 (예외)
4. 라인 내 `` ` `` 카운트가 홀수면 → 백틱 안 (예외)
5. 토큰이 `(` / `[` / `【` / `「` 여는 괄호 그룹 안에 포함되면 → 괄호 안 (기존 한글(한자) 병기 형식의 일부이므로 예외)
6. 토큰 바로 뒤에 `(한글...)` 가 이어지면 → `한자(한글)` 병기 형식 (예외)
7. 토큰 이후 같은 라인에 공백·한자만 있다가 `(한글...)` 이 나오면 → 복합 한자 구절의 말미 병기 (예외)
8. 위 예외에 모두 해당하지 않으면 → **bare 한자 (치환 대상)**
